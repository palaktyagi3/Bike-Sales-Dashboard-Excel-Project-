# Bike-Sales-Dashboard-Excel-Project-
This project presents an interactive Excel dashboard built to analyze bike sales data and uncover patterns in customer behavior.
## 📊 Dashboard Insights
The dashboard provides insights into:
* **Income Analysis**
  Comparison of average income between customers who purchased bikes vs those who didn’t, segmented by gender.
* **Customer Commute Trends**
  Analysis of how commute distance impacts bike purchase decisions.
* **Age Group Analysis**
  Distribution of bike purchases across different age brackets.

## 🎛 Interactive Filters
The dashboard includes slicers for:
* Marital Status
* Region
* Education Level
These allow dynamic filtering for deeper analysis.
